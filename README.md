## Installation

- `git clone https://github.com/awesome-dev877/laravel-vue-crud-starter`
- `cd laravel-vue-crud-starter/`
- `composer install`
- `cp .env.example .env`
- Update `.env` and set your database credentials
- `php artisan migrate`
- `php artisan db:seed`
- `npm install`
- `npm run dev`
- `php artisan serve`
